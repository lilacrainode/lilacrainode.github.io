19.9.8算法学习:冒泡排序#

冒泡排序需要进行n-1遍才能使其全部按顺序排好

打开为知笔记快捷键为ctrl+art+m

问题#

for int i = 1;i<=4;i++   /*C++*/

这样的代码如何在Python中实现？

python实现冒泡排序#

list1 = [3,1,4,5,2]
i = 1
j = 0
temp = 0
while i <= 4:
     i += 1
     while j<=3:
          if list1[j]<list1[j+1]:
               pass
          elif list1[j]>list1[j+1]:
               temp = list1[j+1]
               list1[j+1] = list1[j]
               list1[j] = temp
          else:
               print('something went wrong')
          j += 1   
     j = 0  #要把J还原为0，否则J一直等于3
print(list1)

其中循环之后让j变回0非常重要

3n+1猜想#

卡拉兹猜想 如果n为偶数，那么把他砍掉一半，如果n为奇数，那么把3n+1砍掉一半

n = int(input())
steps = 0
while n != 1:
     if n % 2 == 0:
          n /= 2
     else:
          n = (3*n+1)/2
     steps += 1
print(steps)

如果要想把输入的一行用空格分隔的数字以Int的形式存放到一个list里面，用到如下代码

a = input()
list1 = a.split()
list2 = [int(i) for i in list1]
print(list2)

list2即为得到的全是数字的List
