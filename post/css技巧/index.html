css技巧

  • 将 <a> 标签的css属性中 text-decoration 设置成 none 即可去除 <a> 标签的下划线
  • 这样表示自动居中
  • 如果需要网站页面的主体分为左右两边的话，要把网站主题弄到一个 main 的 div 里面，然后左右两边也分成两个div， leftmain 和 rightmain ，然后把 display 属性设置为 inline-block ，这样可以分成两边 效果：
  • 如果需要鼠标悬停在一个 <a> 标签上面的时候其下面的 <li> 标签改为显示，否则为隐藏，可以这样

li标签的display属性默认为 none

  • rgba 和 rgb 相比多了一个 alpha 属性，可以设置字体的不透明度

RGBA 颜色值是这样规定的：rgba(red, gre en, blue, alpha)。alpha 参数是介于 0.0（完全透明）与 1.0（完全不透明）的数字。

  • margin:0 auto 设置对象上下间距为0，左右自动。(0 auto 0 auto)，需要 居中 的时候一般就用这种方法
  • 要修改 <ul> 中前面的装饰，可以将list-style-type 的值设置为 none
