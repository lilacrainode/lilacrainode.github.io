  • =和<=>的区别是后面的那个可以比较两个NULL值，而如果用前面的那个来比较的话返回的值是NULL
  • SQL语句中如果需要输出一个变量，要用{}将其括起来，否则PHP将会认为是要输出$i后面的字符这个变量
  • mysqli_fetch_row($result);获取select * from t1的结果中的一条
  • mysqli_fetch_row 用索引数组（下标为1，2，3，4，5）获取结果，而mysqli_fetch_assoc 用关联数组的方法获取结果，array(2) { [“id”]=> string(1) “5” [“info”]=> string(6) “雨初” }
  • mysqli_escape_string($link,$temp);可以将$temp字符串里面需要转义的字符全部加上反斜线进行转义
  • mysqli_multi_query 返回的值是false还是true取决于第一条语句有没有成功执行，如果第一条语句成功执行了，无论后面的语句有没有成功执行都返回true
