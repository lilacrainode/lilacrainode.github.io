文章目录
  • webstorm中输入标签然后按 tab 可以自动补全标签
  • ul>li*5(tab) 可以快速插入含有5个li的ul，写了一个li的内容按 enter 可以快速切换到下一个li标签里面
  • ol 列表参数
属性 值 描述
reversed reversed 规定列表顺序为降序(9,8,7…)
start number 规定有序列表的起始值
type 1 A a I i 规定在列表中使用的标记类型（1,2..）(A,B..)(a,b..)
  • <dl> 标签定义了定义列表（definition list），用于结合 <dt> （定义列表中的项目）和 <dd> （描述列表中的项目）。实例：
1
2
3
4
5
6

<dl>
   <dt>计算机</dt>
   <dd>用来计算的仪器 ... ...</dd>
   <dt>显示器</dt>
   <dd>以视觉方式显示信息的装置 ... ...</dd>
</dl>
  • <table> 表示表格， <tr> 表示行，<td> 表示列， <th> 也是列，但是是表头
  • <table> 标签属性
属性 值 描述
border pixels 规定表格边框的宽度
cellspacing pixels% 规定单元格之间的空白（分割线中间的空白）
cellpadding pixels% 规定单元边沿与其内容之间的空白（字和上下两条线之间的距离）
  • <td> 标签中 align 属性规定单元格的 水平对齐方式
  • <td> 标签属性
属性 值 描述
align left right center justify char 规定单元格的水平对齐方式
colspan number 设置单元格 可横跨的列数
