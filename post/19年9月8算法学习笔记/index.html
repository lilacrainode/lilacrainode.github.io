list1 = [3,1,4,5,2]
i = 1
j = 0
temp = 0
while i <= 4:
     i += 1
     while j<=3:
          if list1[j]<list1[j+1]:
               pass
          elif list1[j]>list1[j+1]:
               temp = list1[j+1]
               list1[j+1] = list1[j]
               list1[j] = temp
          else:
               print('something went wrong')
          j += 1   
     j = 0  #要把J还原为0，否则J一直等于3
print(list1)

其中循环之后让j变回0非常重要

3n+1猜想

卡拉兹猜想 如果n为偶数，那么把他砍掉一半，如果n为奇数，那么把3n+1砍掉一半

n = int(input())
steps = 0
while n != 1:
     if n % 2 == 0:
          n /= 2
     else:
          n = (3*n+1)/2
     steps += 1
print(steps)
