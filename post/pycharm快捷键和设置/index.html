Pycharm快捷键和设置

2020年2月3日
| 阅读
  • f12 可以聚焦到上一个使用的窗口
  • pycharm中 ctrl + shift + f12 可以隐藏所有工具窗口
  • alt + shift + e 可以将当前行在console中运行
  • shift + enter 可以 split line ,光标不移到下一行但光标之后的东西移到下一行
  • f4 转到定义
  • ctrl + q 快速查看文档
  • ctrl + alt + t 可以调出surround with窗口
  • ctrl + j 插入模板
  • ctrl + tab 在各窗口之间切换
  • ctrl + f11 加上书签， shift + f11 展示所有书签，如果需要跳转到某个书签直接 ctrl + 书签编号 ，比如如果书签编号是1就按 ctrl + 1 就可以跳转到这个书签
  • 有一个emulate terminal in output console选项，选上了之后就可以不在run窗口而是在终端窗口运行程序
  • f2 和 shift + f2 可以快速定位到错误行
  • alt + shift + c 可以查看最近的修改记录
  • alt + 6 可以调出TODO面板
  • 按住 ctrl 键再按 + 或 - 可以快速进行代码块的折叠/反折叠
  • 选中变量然后 shift + f6 可以重命名变量
  • 将 f12 更换到上一个窗口改成了 ctrl + alt + x
  • 右下角那个 人头像 点开之后选到 syntax 就可以禁止pycharm检查变量的不合命名规则，避免各种奇怪波浪线
  • 将 cyclic expand word 改成了 alt + /
  • view-appearance-toolbar 可以选择是否显示toolbar
  • ctrl + b 可以跳转到定义， ctrl + alt + 左箭头 可以跳转回来
