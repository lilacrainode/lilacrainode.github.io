Linux笔记

2020年1月29日
| 阅读

touch

  • touch 如果文件不存在，创建一个文件，如果文件已经存在，将 文件的修改日期改到最新
  • 在linux下，同一个目录中的文件和目录 不允许重名

mkdir

  • mkdir命令的-p选项可以递归创建目录，即可以创建多重目录

rm

  • rm命令删除文件之后文件不会出现在回收站中
  • rm命令的 -r 选项可以删除文件夹， -f 选项表示强制删除， 不会出现任何提示信息 ，如果需要删除的对象存在就强制删除，如果需要删除的对象不存在也不会报错

tree

  • 单单的tree命令可以展示当前的文件夹下文件和目录的结构，如果是加上了 -d 选项的tree命令则表示仅仅展示当前文件夹下目录的结构，不会显示文件
  • 在gedit中编辑文件的时候，如果当前文件被修改过了，左上角文件名的前面会加上一个星号 * ，ctrl+s之后星号消失

cp

  • 使用cp命令的时候如果文件名不发生变化，第二个参数直接写需要复制到的目录就行，不用写文件名，如果想要复制文件到当前目录，直接
1

cp 源文件 .

就可以

  • cp命令如果怕会覆盖文件，加上 -i 选项，如果可能会覆盖文件就会出现提示
  • cp 命令并不能直接复制目录，如果需要复制目录需要和rm删除命令一样加上 -r 选项
1
2

rm -r test
cp -r a test

mv

  • mv操作和cp命令类似，也存在-i选项，如果可能覆盖文件会出现提示，因为 在终端中对文件的操作是不能回退的 ，所以给文件操作的时候最好加上-i选项

cat && more

  • cat命令中-b选项的含义是对非空输出行编号，-n对输出的所有行编号，这两个选项可以用来快速知道这个文件有多少行代码
  • more命令中空格键用来 显示手册页的下一屏 ，回车用来 显示下一行 b用来回滚一屏，f用来向前滚一瓶，q用来退出， /word 用来搜索word字符串

grep

  • -n选项用来显示对应的行号
1

grep -n aaaa 123.txt //在123.txt中查找aaaa这个字符串并将包含该字符串的每一行的行数显示出来
  • -v 选项用来展示所有不包含该字符串的行（相当于取反）
  • 如果想要 忽略大小写 ，要加上 -i 选项

重定向

  • > 一个大于号表示输出，会 覆盖文件原有的内容 ，（可以联想fprintf中的w选项）
  • » 两个大于号表示追加，会 将内容追加到已有文件的末尾 (可以联想fprintf中的a选项)

ls

1

ls -lh //h选项表示可以将文件的大小以人类可读的形势展示（kb,mb,gb）,还有
  • 用echo命令来创建文件可以让文件中有内容，如果用touch命令来创建文件则会创建一个什么都没有的空文件

管道|

  • 管道的左边塞东西（写），右边取东西（读）
1

ls -lh | more // 这个命令将ls -lh的结果more，如果ls -lh返回的结果过多的话这样做可以避免ls -lh的结果在一屏显示的尴尬

shutdown

  • 如果不指定选项和参数，直接shutdown一个命令，表示 1分钟 之后 关闭电脑
  • -r 选项用来表示重启电脑
  • 在执行shutdown指令之后如果要取消shutdown要输入
1

shutdown -c
  • 常用命令实例：
 1
 2
 3
 4
 5
 6

# 重新启动操作系统，其中 now 表示现在
$ shutdown -r now
# 立刻关机，其中 now 表示现在 
$ shutdown now
# 系统在今天的 20:25 会关机 
$ shutdown 20:25

$ shutdown +10

$ shutdown -c
  • 用ssh操作远程服务器的时候一般不会使用关机命令，一般都会加上-r命令，因为服务器关机了启动就很麻烦

网络

  • 查看IP地址-右上角设置-网络-有线

网卡是一个专门负责网络通讯的硬件设备

IP地址是设置在网卡上的地址信息

  • ifconfig可以查看/配置计算机当前的网卡配置信息
  • 查看网卡对应的IP地址：
1

ifconfig | grep inet
  • 一台计算机中有可能会有一个 物理网卡 <! –（也可能多个？）—>和 多个虚拟网卡 ，在linux中物理网卡的名字通常以 ensXX 来表示

ssh基础（重点）

  • SSH客户端是一种使用 secure shell(ssh) 协议连接到远程计算机的软件程序，数据传输是 加密 的，可以防止信息泄露，数据传输还是 压缩 的可以提高传输速度

ssh使用

1

ssh [-p port] user@remote
  • user是在远程机器上的用户名，如果不指定的话默认为当前用户
  • remote是远程机器的地址，可以是IP或者是域名，或者是别名（后面会提到）
  • port 是 ssh server 监听的端口 如果不指定， 默认为22端口

提示

  • 使用 exit 退出当前用户的登录
  • 在linux和mac中ssh默认安装，但是如果要在window下使用ssh可以安装 puTTY或者xShell 客户端软件即可
  • 在工作中SSH服务器的端口号很可能不是22，这种情况下就需要使用 -p 选项，指定正确的端口号，否则无法正确连接到服务器

scp

  • scp如果需要指定端口，参数为 -P ，注意P要大写
  • ftp的端口是21，ssh的端口是22

ssh高级

  • 免密码登录
  • 配置别名 视频链接
  • 在用户根目录下面有一个隐藏的 .ssh 文件夹
  • ssh-keygen 命令会生成sshkey，然后
1

ssh-copy-id 用户名@远程服务器地址

就可以让远程服务器记住sshkey,之后再登录可以免密码登录

非对称加密算法

  • 使用 公钥 加密的数据，需要使用 私钥 解密
  • 使用 私钥 加密的数据，需要使用 公钥 解密
  • id_rsa.pub是公钥，id_rsa是私钥
  • ssh-copy-id会将公钥发到对方服务器上面，到时候就可以直接用公钥和私钥进行通信，不需要再输入密码
  • 如果要给远程服务器加上别名，要在ssh文件里面加上一个 config 文件，然后里面写上
1
2
3
4

Host huawei
    HostName 119.3.214.112
    User root
    Port 22

以后如果想要登录就不用输入 ssh -p 22 root@119.3.214.112 了，直接输入

1

ssh huawei

就可以了

用户权限相关命令

  • 为了方便用户管理，提出了 组 的概念，在实际应用中，可以预先针对 组 设置好权限，然后 将不同的用户添加到对应的组中 ，从而不用 依次为每一个用户设置权限
  • r表示可读，w表示可写，x表示可执行
  • ls-l之后看到的权限中 第一组权限对应的是文件的拥有者的权限 ， 第二组权限对应的是文件拥有者对应的组的权限 ， 第三组权限对应的是其他用户拥有的权限
  • ls-l执行的结果中第一个字符表示这个文件 的属性 ，然后是 文件的权限 ，然后是 其硬链接数 ，一个目录的子目录越多，他的硬链接数越多。然后是该文件拥有者的用户名，然后是该文件拥有者的组名，然后是文件大小，然后是文件修改时间，然后是文件名

硬链接（hard link，也称链接）就是一个文件的一个或多个文件名。再说白点，所谓链接无非是把文件名和计算机文件系统使用的节点号链接起来。因此我们可以用多个文件名与同一个文件进行链接，这些文件名可以在同一目录或不同目录。 [硬链接资料(百度百科)] (https://baike.baidu.com/item/%E7%A1%AC%E9%93%BE%E6%8E%A5)

  • 在linux中很多时候会出现组名和用户名相同的情况
  • chomd +/- 可以用来修改文件/目录的权限
  • 在linux中如果要执行某个文件，要 ./01.py 这样才能执行
  • 一个目录如果没有x权限，这个目录不能用来执行终端命令，比如如果test这个文件夹没有x权限， cd test 这个命令将不能执行，将会提示权限不够

超级用户

  • su是 substitute user 的缩写，表示 使用另一个用户的身份
  • sudo命令用来以其他身份来执行命令，预设的身份为root
  • 使用 sudo 时，必须先输入密码， 之后有 5分钟的有效期限 ，在期限内不需要再输入密码，超过期限则必须重新输入密码

组管理 终端命令

  • 组信息保存在 /etc/group 这个文件中
  • 在根目录etc目录下保存的绝大多数文件都是跟 系统配置 有关的
  • 要添加组的话要
1

groupadd 组名  # 要注意要sudo才有权限添加组

之后要确认是否添加成功可以

1

cat /etc/group  # 如果添加成功的话可以在最后一行看到添加的组信息
  • chgrp -R 组名 文件/目录名 递归修改文件/目录的所属组（要sudo），这个命令是由 ch(change) 和 grp(group) 合成的，可以以change group来记忆

用户管理 终端命令

  • 添加新用户：
1
2
3
4

useradd -m -g 组 新建用户名    # 添加新用户
# -m表示自动建立用户家目录 （如果忘记输入，就需要我们自己给这个用户创建家目录了）
# -g表示指定用户所在的组，否则会建立一个和用户名同名的组
passwd 用户名    # 设置用户密码，如果是普通用户，直接用passwd可以修改自己的账户密码
  • 如果useradd的时候没有添加-m选项，默认不会创建/home/用户名 的目录，所以如果忘了加 -m 选项会变得非常麻烦，建议删掉这个用户再重新添加一个
  • 删除用户：
1

userdel -r 用户名 # -r选项会自动删除用户家目录

建议使用userdel的时候都要加上-r选项

  • 使用userdel可能会提示
1

userdel: zhangsan 邮件池 (/var/mail/zhangsan) 未找到

这是因为这个文件本来就不存在，这时候zhangsan这个用户已经被成功删除了，忽略这条提示信息即可

（第72集）uid是存放在 /etc/passwd 这个文件下面的 gid在 /etc/group 这个文件下查看

在passwd文件中以 冒号: 来分割信息。 第二列为 x 表示该用户是有密码的，但是密码是加密的，然后是uid,然后是该用户所在组的uid(是uid吗？)，然后是这个用户的全名，如果为空（出现了两个冒号连在一起的情况），说明这个用户以用户名作为全名，然后是该用户的家目录，最后一项是（待添加）

bin和sbin

  • 一句话概括：bin存储的是普通可执行文件，sbin存储的是和系统管理相关的可执行文件
  • 在linux中，绝大多数可执行文件都是保存在 /bin,/sbin,/usr/bin,/usr/sbin 中的
  • /bin(binary) 是二进制执行文件目录，主要用于具体应用
  • /sbin （system binary） 是系统管理员专用的二进制代码存放目录，主要用于系统管理
  • /usr/bin （user commands for applications）后期安装的一些软件
  • /usr/sbin （super user commands for applications） 超级用户的一些管理程序

提示：

  • cd 这个终端命令是 内置在系统内核中的 ，没有独立的文件，因此用 which 无法找到 cd 命令的位置

跟进程信息相关命令

进程信息 （所谓进程，就是当前正在执行的一个程序）

ps 选项功能说明

选项 含义
a 显示终端上所有进程，包括 其他用户的进程
u 显示进程的详细状态
x 显示没有控制终端的进程
